Przy otwarciu czystego dokumentu zacznij od:

W arkuszu html wpisz znak "!" a następnie tab.
Po tym pamiętaj by zmnienić język w jakim będzie wyświetlany tekst strony, tą możliwość znajdziesz tutaj:
< html lang="pl"> domyślnie nie będzie to "pl" .
W nagłówku < head > możesz wpisać:
< link href="style.css" rel="stylesheet" type="text/css"> , jest to odnośnik do arkusza stylów css w którym to będziemy dbali o to jak wyglda nasza strona.

Struktura pisania w HTML- początek

Ogólnie html jest prosty jak wydanie 2mld zł na tvp

Podstawowa wiedza:

< body > wszystko co chcemy wyswietlic na stronie musi być umieszczone w tym znaczniku

< main > jest to główna część/treść naszej strony, ona stanowi priorytet przy wyświetlaniu na przeglądarkach

< header > może stanowić nagłówek dokumentu, logo itp, czyli element mający rzucać się w oczy, wewnątrz niego umieszczamy zazwyczaj < h1 > NAGŁÓWEK < /h1 >

-dodając header pod main będzie on rozpatrywany przez przeglądarkę

-dodając go przed < /main > wyszukiwarki nie będą go analizować, przykładowo można go użyć jako przyscisk powrotu do strony głównej

< section > sekcje dokumentu (części podziału strony)

< article > samodzielna treść/część strony

< h1 > nagłówek pierwszego poziomu, najniższym jest h6

< p > paragraf/akapit, tekst główny umieszczony np. pod nagłówkami

< footer > stopka dokumentu/strony, najlepiej przed < /main > tak jak w przypadku header

Jest to stopka strony, kiedyś coś tu napiszę. Wróć na górę strony
